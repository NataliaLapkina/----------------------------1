MVP-приложение «Личный кабинет нутрициолога»
1. Главная идея про

Создавайте маленькие, аккуратные и эмоциональные безоп.

Это не фитнес-трекер.
Это не приложение для жёсткого контроля питания.
Это не система оценок

Главная задача приложения — помочь клиенту спокойно наблюдать за собой

Помещение должно быть простым, теплым, современным и удобным для телефона.

2. Технический MVP

Создать первую версию, в которой:

посмотреть рекомендации нутрициолога;
посмотреть комментарии специалиста.
Нутрициолог может:
зарегистрироваться;

смотреть ежедневные записи клиента;
оставляет комментарии;
Рекомендации по эксплуатации.

6. Роли пользователей

В приложении есть два ролика:

1. лж

Пользователь, который выполняет ежедневные записи и получает рекомендации.

2. Нутрициолог

Специалист, который ведет клиентов, записывает и оставляет комментарии.
